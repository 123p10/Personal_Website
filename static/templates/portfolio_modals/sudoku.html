March 2019

Sudoku Solver Repository

Many sudoku solvers on the internet use a bruteforce method. I thought it would be an interesting challenge to develop a sudoku solvers that utilized various of my own algorithms rather than using standard techniques or some inefficient bruteforce technique. I developed a sudoku solver which can solve fairly complex boards in fractions of seconds. The program was able to solve,check and render the board shown above into the solution below in just over 8ms. This program was developed in Java.
