Lego 3D Printer

October 2019 - December 2019

3D Printer Report

In my first term of Mechatronics Engineering at Waterloo we were tasked with designing and building a term project with these lego robotics kits. My group and I decided that we wanted to build a 3D printer. Through numerous design iterations we were able to come up with a functioning 3D printer which could produce an object through a custom file format.
The full process is detailed in the report linked above.
